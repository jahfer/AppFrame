AppFrame is a lightweight and extendable framework for iterative applications. Based on the runtime structure of the XNA framework.

AppFrame Reference
Constructor
  • AppFrame(fps:number, initialized?:boolean)
    First argument will set the rate at which the main loop will run in frames per second.
    Second argument is optional and sets whether the initialized variable is set to true or false. Default set to false.
Commands
  • start()
    Calls the initialize function and starts the AppFrame runtime functions once initialized has been set to true.
  • stop()
    Stops the AppFrame loop and calls the exit function
Runtime
  • initialize()
    Runs once when start() is called. Mainly used for initialization and loading of application.
  • update()
    Part of the main loop. Runs each iteration before the draw() function. Mainly used to update instances of the application.
  • draw()
    Part of the main loop. Runs after update(). Mainly used for all visual render/draw functions.
  • exit()
    Runs once when stop() is called. This is the last function to run before the application stops running.
Variables
  • initialized
    Type boolean. If this is set to false, the main loop will not run until it has been set to true.
